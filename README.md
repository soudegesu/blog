# blog
github pages with [Hugo](https://gohugo.io/).

## Preconditions
* [Hugo](https://gohugo.io/)
    * v0.59

## Quick Start

* create new post file

```
hugo new post/articlefile.md
```

* build posts with local server. (including draft)
```
make serve
```

* build posts for publish article (exclude draft)
```
make pub
```
